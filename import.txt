Tải về tailwindcss
Bước 1: npm install tailwindcss @tailwindcss/vite
Bước 2: Thay đổi nội dung tệp index.css thành @import "tailwindcss";
Bước 3: Thêm  import tailwindcss from '@tailwindcss/vite' và  plugins: [react(), tailwindcss()], vào  vite.config.js

tạo file db.json tạo từ my-app( cho data vào) lưu file ngang hàng vs src
sau đó chạy lệnh npx json-server --watch db.json --port 3001
sau đó dán host lưu trữ vào và sử dụng http://localhost:3001/students
